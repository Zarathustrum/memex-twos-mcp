# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2024-12-29

### Added
- **Hybrid Search**: Combines BM25 lexical search with semantic vector search for better conceptual query matching.
- **Incremental Ingestion**: Content-hash based change detection for faster database updates (10-100x faster).
- **Entity Extraction**: Integrated spaCy NER for accurate people extraction (precision ~90%).
- **Two-Phase Retrieval**: Optimized search with result caching.
- **Data Grooming**: AI-assisted tools for duplicate removal and entity normalization.
